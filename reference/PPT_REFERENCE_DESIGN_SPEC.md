# PPT Reference Design Mapping

The PPT generator is designed to match the uploaded Monthly Inventory Report reference deck structure:

1. Inventory Performance Report / Executive Summary
   - Management Snapshot
   - KPI cards
   - Key Message
   - Management Highlights
   - Footer with system name, internal reporting note, generated time, and page number

2. Monthly KPI Overview
   - KPI cards
   - Interpretation section
   - Attention Needed section

3. Sample-In Transaction Trend
   - 12-month sample-in entries chart
   - Trend Insight block

4. Inventory Composition
   - Product Sample Category
   - Product Status Summary
   - Total Records / Usable / Expiring / Expired focus block

5. Recent Sample Movement
   - Recently Received Samples table
   - detailed-records tip

6. Actions and Follow-up
   - Released Samples This Month
   - Recommended Actions
   - Monthly Release Summary

Design details:
- Green header/footer bands
- White report background
- KPI cards with icon circles
- Rounded cards/panels
- Management-report tone
- Minimal tables and charts for presentation readability
